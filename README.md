# Projeto Integrador - LifeTrack

Portfólio e repositório monorepo do sistema **LifeTrack**. A documentação operacional oficial fica em [LifeTrack/README.md](LifeTrack/README.md).

## Estrutura

```text
Projeto-Integrador/
├── .vscode/        # Configurações do workspace no VS Code
├── LifeTrack/      # Aplicação principal, documentação e guias
└── README.md       # Visão geral da raiz do repositório
```

## Leitura recomendada

- [LifeTrack/README.md](LifeTrack/README.md) é o guia oficial para rodar o projeto.
- [LifeTrack/GUIA_DE_COMMITS.md](LifeTrack/GUIA_DE_COMMITS.md) traz o padrão de commits do time.
- [LifeTrack/backend/README.md](LifeTrack/backend/README.md) e [LifeTrack/frontend/README.md](LifeTrack/frontend/README.md) são referências curtas por área.

## Observação

A raiz foi mantida intencionalmente enxuta para facilitar a apresentação do projeto como portfólio. Para executar o sistema, siga o guia oficial dentro de `LifeTrack/`.
