# LifeTrack Project 🚑

Repositório principal do sistema LifeTrack, um software profissional de gestão e despacho de ambulâncias e ocorrências de emergência.

## 🏗️ Estrutura do Repositório (Monorepo)

O projeto está organizado de forma modular para clara separação de responsabilidades:

- 📂 **`backend/`**: Contém a API principal do sistema (Lógica de negócios, acesso a banco de dados e endpoints).
- 📂 **`frontend-angular/`**: Aplicação frontend atual (SPA desenvolvida em Angular 19+). **(Em desenvolvimento)**
- 📂 **`frontend-react-legacy/`**: Código legado em React (Mantido provisoriamente para consulta durante a migração arquitetural).
- 📂 **`docs/`**: Documentação técnica detalhada:
  - `INSTALACAO.md`: Guia de configuração e build.
  - `RECOMENDACAO_OPERADOR.md`: Manuais de processos de negócio.
  - `GITIGNORE_EXPLICACAO.md`: Regras de versionamento.

## 🚀 Como Iniciar

Consulte a pasta `docs/` ou acesse os respectivos diretórios `backend/` e `frontend-angular/` para instruções específicas de execução de cada ecossistema.
